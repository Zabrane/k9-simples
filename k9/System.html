11 System

This chapter describes the system settings and functions. 

System
\l a.k See load
\t:n x See timing
\u:n
\v     See variables
\w     See memory
\cd x  See cd

11.1 load ⇒ \l a.k

Load a text file of k9 commands. The file name must end in .k. 

 \l func.k
\l func.k
 \l func.k7  / will error as not .k
error: nyi

11.2 timing ⇒ \t

List time elapsed 

 \t ^(_1e7)?_1e8
227

11.3 variables ⇒ \v

List variables 

a:1;b:2;c:3
 \v
[v:`a`b`c]

11.4 memory ⇒ \w

List memory usage 

 \w
0
 r:(`i$10e6)?10
 \w
2097158

11.5 cd ⇒ \cd x

Change directory (cd) into x 

 \cd scripts
