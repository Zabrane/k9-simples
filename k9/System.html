11 System

This chapter describes the system settings and functions.

11.1 Display ⇒ \k

 m:(100 101 102;3;14 15);d:`a`b!(1 2;3 4)
 \k 0
 m
100 101 102
3          
14 15      

 d
a|1 2
b|3 4

 \k 1
 m
(100 101 102;3;14 15)
 d
[a:1 2;b:3 4]

11.2 Load File⇒ \l

Load a text file of k9 commands. The file name must end in .k. 

 \l func.k
\l func.k
 \l func.k7  / will error as not .k
error: nyi

11.3 Variables ⇒ \v

List variables 

a:1;b:2;c:3
 \v
[v:`a`b`c]
